🎨 Design System

Color Palette

Blue
Green
Orange
Red

Key Features

  • Driver Management: Registration, authentication, status tracking
  • Real-time Updates: WebSocket connections for live updates
  • GPS Tracking: Location-based driver assignment
  • Customer Experience: Order tracking and notifications
  • Analytics: Performance metrics and insights

Technology Stack

  • Backend: Node.js, Express, Socket.io
  • Database: PostgreSQL
  • Frontend: React.js (Progressive Web App)
  • Maps: Google Maps API
  • Notifications: Firebase Cloud Messaging
